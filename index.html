What is Cupertino.css?

Cupertino.css is a classless CSS framework inspired by Apple's design language. It provides beautiful styling without requiring any CSS classes in your HTML.

How to use it?

Simply include the CSS file in your HTML and write semantic markup. The framework will automatically style your content with Apple-inspired design.

Here's how you can display code blocks:

<html>
<head>
    <link rel="stylesheet" href="cupertino.css">
</head>
<body>
    <h1>Hello, World!</h1>
    <p>No classes needed.</p>
</body>
</html>
